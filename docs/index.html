Quick Setup

Install dependencies and start using AureaTools in seconds.

1

Clone the Repository

Get the latest source code from GitHub to your local machine.

git clone https://github.com/js-victr/aurea-tools.git
cd aurea-tools
2

Install Dependencies

Install the required Python modules in development mode using pip.

pip install -e .
3

Run the App

Launch the interactive console in your shell.

python -m aurea

Instalação & Configuração Rápida

Instale dependências e ative a suíte AureaTools em segundos no Windows, Linux ou Termux (Android).

1

Clonar o Repositório

Baixe a versão mais recente do código-fonte do GitHub Pages para a sua máquina local.

git clone https://github.com/js-victr/aurea-tools.git
cd aurea-tools
2

Instalar Dependências

Configure as bibliotecas necessárias do Python utilizando o `requirements.txt` integrado.

pip install -r requirements.txt
3

Iniciar a Suíte

Execute o módulo principal para abrir a interface gráfica CLI traduzida automaticamente.

python -m aurea
